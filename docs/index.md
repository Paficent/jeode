# Jeode

Mod loader & framework for the PC version of **My Singing Monsters**. Jeode lets you build unique
creations that leverage asset replacements, [Lua](https://www.lua.org/manual/5.1/) scripts, and native DLL mods.

Jeode is completley open source under the [BSL-1.0 License](https://github.com/Paficent/jeode/tree/main?tab=BSL-1.0-1-ov-file). The code is hosted on [GitHub](https://github.com/paficent/jeode) and mirrored to [Codeberg](https://codeberg.org/paficent/jeode).
